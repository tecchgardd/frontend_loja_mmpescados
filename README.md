# MM Pescados — Loja Front

Front separado da loja/e-commerce da MM Pescados.

## Rodar local

```bash
npm install
npm run dev
```

## Backend

Este front consome o mesmo backend do painel admin usando o `.env`:

```env
VITE_API_URL=http://localhost:3333/api
```

## Rotas principais

- `/loja/login`
- `/cadastro`
- `/loja`
- `/loja/pedidos`
- `/loja/conta`
